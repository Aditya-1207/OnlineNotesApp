# OnlineNotesApp
Backend Implementation of RESTful CRUD API for Notes making using Spring Boot, MySQL, JPA and Hibernate

# REST CRUD APIs

GET /api/notes

POST /api/notes

GET /api/notes/{Id}

PUT /api/notes/{Id}

DELETE /api/notes/{Id}

We can test them using postman or any other rest client.
